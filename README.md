# My WordPress Site

This repository contains the code for my custom WordPress site, including a custom theme and plugin.

## Theme

The custom theme is located in wp-content/themes/my-custom-theme.

## Plugin

The custom plugin is located in wp-content/plugins/my-custom-plugin.

## CI/CD

The CI/CD pipeline is configured using GitHub Actions and is defined in .github/workflows/ci.yml.

## Setup

1. Clone the repository.
2. Set up a local development environment.
3. Push changes to the main branch to trigger the CI/CD pipeline.
